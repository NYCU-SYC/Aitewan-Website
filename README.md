# Aitewan Website

Official website project for AItewan BioMedical Technology.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

```bash
npm run lint
npx tsc --noEmit
```

## Build

```bash
npm run build
npm run start
```
